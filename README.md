Čítačka na bezpečnostnú bránu pomocou ESP
Popis projektu

Projekt sa zaoberá návrhom a realizáciou inteligentného systému na ovládanie bezpečnostnej brány pomocou mikrokontroléra ESP32. Cieľom projektu je vytvoriť jednoduchý a spoľahlivý prístupový systém, ktorý umožňuje otváranie a zatváranie brány na základe identifikácie používateľa alebo prostredníctvom vzdialeného ovládania cez webové rozhranie.

ESP32 v tomto projekte slúži ako hlavná riadiaca jednotka, ktorá spracúva vstupné údaje, riadi výstupné zariadenia a zabezpečuje bezdrôtovú komunikáciu prostredníctvom WiFi siete. Program je vytvorený v prostredí Arduino framework a je napísaný v jazyku C++.

Použité komponenty
•  ESP32 vývojová doska
•  RFID čítačka MFRC522 (RC522)
•  RFID karta / RFID čip 
•  LCD displej 16×2 s I2C rozhraním
•  Servomotor
•  breadboard
•  Prepojovacie vodiče
